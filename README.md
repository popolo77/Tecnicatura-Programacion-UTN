### 🔹 PRIMER CUATRIMESTRE
- [x] 🟢 **Programación I** (128 hs totales)
  * **Contenidos mínimos:** Introducción a los distintos paradigmas de programación, Programación Imperativa, Concepto de algoritmo, Estrategias de Resolución de problemas, Concepto y tipos de datos, Variables y constantes, Estructuras de Control Básicas, Estructuras de Datos, Abstracciones con procedimientos y funciones, Recursividad, Algoritmos de Búsqueda, Recorrido y Ordenamiento.

- [x] 🟢 **Arquitectura y Sistemas Operativos** (64 hs totales)
  * **Contenidos mínimos:** Arquitectura de computadoras, Estructura, características y clasificación de Sistemas Operativos, Planificación e hilos en Procesos, Comunicación y Sincronización entre Procesos, Gestión de Memoria, Sistemas de Archivos, Gestión de Entrada/Salida e Interrupciones, Introducción a las redes de datos y Virtualización.

- [x] 🟢 **Matemática** (64 hs totales)
  * **Contenidos mínimos:** Conjuntos (Pertenencia, inclusión, operaciones básicas, producto cartesiano, relaciones binarias), Sistema binario, Matrices y operaciones, Lógica, Álgebra de Boole, Grafos y árboles.

- [ ] 🟡 **Organización Empresarial** (64 hs totales) *[Regular]*
  * **Contenidos mínimos:** Teoría general de Sistemas, Información, Conocimiento y Toma de Decisiones, Organización y Empresa, Sistemas administrativos y contables, Planificación, organización, dirección y control, Gestión por Procesos.

---

### 🔹 SEGUNDO CUATRIMESTRE
- [ ] ⚪ **Programación II** (128 hs totales)
  * **Contenidos mínimos:** Principios de la Orientación a Objetos, Clases (Concepto, elementos, atributos y métodos, implementación), Constructores, Enumeraciones, Arreglos y colecciones, Acceso a datos, Gestión de altas, bajas, modificación y consultas (ABMC), Introducción a Servicios WEB.

- [ ] ⚪ **Probabilidad y Estadística** (64 hs totales)
  * **Contenidos mínimos:** Datos Estadísticos, Análisis combinatorio, Estadística descriptiva, Teoría de las Probabilidades, Variables aleatorias, Distribuciones de Probabilidad, Teoría del Muestreo y Estimación.

- [ ] ⚪ **Bases de Datos I** (64 hs totales)
  * **Contenidos mínimos:** Modelo conceptual de datos (Entidades, Atributos, Relaciones), Modelo Relacional, Sistema de Gestión de Bases de Datos Relacional, Diseño de base de datos, Formas normales y Normalización, Lenguajes de acceso a Bases de Datos Relacionales, Integridad de Datos, Transacciones y acceso concurrente.

- [ ] ⚪ **Inglés I** (64 hs totales)
  * **Contenidos mínimos:** Dimensión lingüística (agencia, nominalización, temporalidad, voz), Dimensión sociolingüístico-discursiva (géneros del dominio académico-profesional), Dimensión estratégica (paratextos, traducción léxica), Dimensión socio-cultural.

---

### 🔹 TERCER CUATRIMESTRE
- [ ] ⚪ **Programación III** (128 hs totales)
  * **Contenidos mínimos:** Aplicaciones WEB, Servicios WEB, Arquitectura de las aplicaciones WEB, Desarrollo de aplicaciones para Internet, Conceptos de Servidor WEB y Cliente WEB, Diseño de aplicaciones Mobile (Responsive, First Mobile, Nativas), Introducción a Tecnologías Mobile, Tecnologías Front-End y Back-End, Validación de controles, Transmisión de datos entre formularios, Mapeo objeto-relacional (ORM).

- [ ] ⚪ **Bases de Datos II** (64 hs totales)
  * **Contenidos mínimos:** Integridad de datos, Transacciones y acceso concurrente, Optimización de consultas, Gestión de índices y vistas, Administración y Seguridad, Mantenimiento de una Base de Datos, Normalización avanzada y eliminación de redundancia, Otras Bases de Datos (NoSQL) y nuevas formas de almacenamiento, Base de Datos para dispositivos móviles.

- [ ] ⚪ **Metodología de Sistemas I** (64 hs totales)
  * **Contenidos mínimos:** Proceso de desarrollo de software (principales componentes), Herramientas empleadas en el proceso de desarrollo, Técnicas y herramientas de gestión de proyectos de software, Ejecución y análisis de resultados generados por las operaciones de pruebas.

- [ ] ⚪ **Inglés II** (64 hs totales)
  * **Contenidos mínimos:** Dimensión lingüística avanzada, Dimensión sociolingüístico-discursiva (géneros científico-académicos complejos), Dimensión estratégica (paráfrasis, circunloquio), Dimensión socio-cultural intercultural.

---

### 🔹 CUARTO CUATRIMESTRE
- [ ] ⚪ **Programación IV** (128 hs totales)
  * **Contenidos mínimos:** Autenticación por medios externos, Aplicación de base de datos en tiempo real, Programación del lado del cliente, Programación del lado del servidor y comunicación con el cliente, Arquitectura Cliente-Servidor.

- [ ] ⚪ **Metodología de Sistemas II** (64 hs totales)
  * **Contenidos mínimos:** Introducción a los patrones de diseño y desarrollo de software, Buenas prácticas en el proceso de implementación de software, Técnicas de optimización del ciclo de desarrollo, Herramientas de verificación y validación, Herramientas de repositorios de software.

- [ ] ⚪ **Introducción al Análisis de Datos** (32 hs totales)
  * **Contenidos mínimos:** Introducción al Análisis de Datos, Herramientas de análisis de datos, Modelado.

- [ ] ⚪ **Legislación** (32 hs totales)
  * **Contenidos mínimos:** Introducción al Derecho y a las Sociedades, Legislación laboral, Derecho Informático y propiedad intelectual, Responsabilidades civiles, penales y profesionales en desarrollo de software, Actuación en procesos judiciales, Seguridad de la información.

- [ ] ⚪ **Gestión de Desarrollos de Software** (64 hs totales)
  * **Contenidos mínimos:** Software (Proceso, Producto, Ingeniería de Requerimientos), Propuesta de Proyecto, Alcance, Criterios de aceptación, Estimación de tiempos y costos, Métricas de Software, Calendarización (cronograma de hitos), Artefactos técnicos requeridos, Reportes e Indicadores, Manuales e Instructivos.

- [ ] ⚪ **Trabajo Final Integrador** (95 hs totales)
  * **Enfoque:** Planteamiento de soluciones a problemas de desarrollo de un producto de software en contextos reales o simulados, incorporando elementos originales. Producto final definible como desarrollo aplicable al entorno con posibilidad de transferencia al medio o resolución de problemas locales.
